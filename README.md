## 🩹 Prevención de Lesiones

### Tobillos Inestables
- La inestabilidad se corrige con **propiocepción** (balance en una pierna) y **fortalecimiento de peroneos** (eversión con banda)
- Hacer ejercicios de tobillo **descalza** para mejor activación
- Si un tobillo se siente inestable antes de correr → 2-3 min de balance como activación pre-run

### Tendón Patelar (Rodilla)
- Se previene con **carga excéntrica progresiva** (Spanish squats, single-leg eccentric squats)
- Si vuelve el dolor → reducir volumen de running, aumentar ejercicios excéntricos
- Si persiste más de 3 días → consultar kinesiólogo

### Fascitis Plantar (Talones)
- Rolling con pelota de tenis después de cada run — **obligatorio**
- Estirar gemelos y sóleo **todos los días** (30s/lado mínimo)
- Calzado adecuado con buen soporte de arco

### Regla del 10%
- Nunca aumentar más del 10% del volumen semanal respecto a la semana anterior
- Si algo duele, **bajar volumen inmediatamente**
- Mejor perder un entrenamiento que perder la carrera

## 🧠 Notas del Plan

### ¿Por qué 9 semanas es suficiente?
Vale no es principiante — ya corrió un 21K (~2:20) y un 10K (1:10). Su cuerpo tiene memoria muscular. Los 5 meses sin correr significan que necesita reactivación, pero no partir de cero.

### Sobre el ritmo 6:30/km
Su 21K anterior fue ~6:40/km (2:20 en 21K). Un pace de 6:30/km es ambicioso pero realista. **Plan B:** si el entrenamiento no fluye como se espera, correr a 7:00/km y disfrutar la experiencia (sub 2:27, que sigue siendo un tiempazo).

### Sobre la escalada
La escalada es excelente cross-training (upper body + core + grip). El único riesgo es la fatiga de piernas. Si las piernas se sienten pesadas antes de un run → priorizar running esa semana. La escalada del jueves puede competir con el long run del fin de semana — monitorear.

### Sobre volver después de 5 meses
Las primeras 2-3 semanas van a ser las más difíciles psicológicamente. Va a sentir que corre lento y se cansa rápido — **es completamente normal**. El fitness cardiovascular vuelve antes que el musculoesquelético — por eso la fuerza es clave en paralelo.

---

## ⚡ Reglas de Oro

1. 🛑 **Si algo duele, parar.** Mejor perder un entrenamiento que perder la carrera.
2. 🐢 **Los días easy son EASY.** No correr rápido porque se siente bien.
3. 💧 **Hidratarse siempre.** Especialmente en runs de +45 min.
4. 😴 **Dormir 7-8 horas.** El descanso es donde el cuerpo se adapta.
5. 🧗 **La escalada está bien** — pero si las piernas están cargadas, priorizar running.
6. 🧠 **Confiar en el proceso.** 9 semanas es suficiente para alguien que ya corrió 21K.
